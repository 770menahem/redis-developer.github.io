Tools

GUI tools for Redis#

Find the best Graphical User Interfaces to interact with your data and, manage and monitor your Redis Database.

RedisInsight
RedisInsight provides an intuitive and efficient GUI for Redis
Redis Data Source for Grafana
The Redis Data Source for Grafana allows users to connect to the Redis and build Grafana dashboards.
RDM
Cross-platform open source database management
Redsmin
Redsmin offers a real-time, atomic, performant administration and monitoring interface for Redis.

Others#

RIOT - Get data in and out of Redis
Redis Input/Output Tools (RIOT) is a set of import/export command line utilities for Redis
Memtier Benchmark
memtier_benchmark is a command line load generation and bechmarking tool developed by Redis Labs.
RediSearch plugin for Wordpress
Use RediSearch to power your Wordpress Search capabilities.
Redis Sitesearch
redis-sitesearch is a package that indexes web sites into RediSearch.
Redis MONITOR command
Useful command to capture command processed by Redis Server
